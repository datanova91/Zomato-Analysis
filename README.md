# Zomato-Analysis
A detailed insight into restaurant ratings, customer votes, and cost trends across dining categories.

CASE STUDY

Objective

To evaluate restaurant ratings, cost patterns, and food type performance.

Key Insights

1. Dataset Summary

-Total Restaurants: 145

-Total Votes: 39,192

-Average Cost for Two: ₹418

-Restaurant Types: Dining, Cafes, Buffet, Others

2. Top-Rated Restaurants

Highest ratings observed in:

-Onesta

-San Churro Cafe

-Corner House Ice Cream

-Meghana Foods

3. Votes by Type

-Dining → 51% of total votes

-Cafes → 23.9%

-Buffet & Others → Have fewer votes

4. Cost Analysis

-Buffets have the highest average cost for two.

-Dining restaurants offer mid-range pricing and highest engagement.

Business Recommendations

-Dining restaurants should leverage high engagement for upselling combos.

-Cafes can introduce student-friendly pricing.

Lower-rated restaurants should revamp menu & service quality.
